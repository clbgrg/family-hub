# Family Hub

DIY self-hosted family calendar + chore system. Raspberry Pi 5 + portable touchscreen, runs on your home WiFi. No cloud, no subscriptions. Built on a fork of [Skylite-UX](https://github.com/Wetzel402/Skylite-UX).

## Quick orientation

- **New here?** Read `CLAUDE.md`, then `docs/build-order.md`.
- **Building hardware?** `docs/hardware.md` + `docs/installation.md`.
- **Writing features?** `docs/features-to-build.md` + `docs/settings-spec.md`.
- **Decisions still open?** `OPEN-QUESTIONS.md`.

## Stack

Nuxt.js + TailwindCSS / Node.js / PostgreSQL, deployed via Docker Compose. Chromium kiosk on the Pi.

## Cost

~$250–290 required parts, ~$350–380 fully loaded. vs. Skylight commercial $300–600 + subscription.

Planning complete, build not started. The Skylite-UX fork has not yet been cloned into this repo.
