# Recommended Build Order

Suggested order to get to a working system as fast as possible.

| Phase | What to do |
|---|---|
| Phase 1 — Hardware | Buy and assemble Pi 5 + UPERFECT screen. Flash Pi OS. Get SSH working. Test screen and touch input. |
| Phase 2 — Skylite-UX running | Install Docker. Clone and start Skylite-UX. Verify access from a phone. Add iCal URL. Create family accounts. |
| Phase 3 — Kiosk mode | Set up Chromium kiosk on the Pi screen. Configure boot-to-app. Test sleep/wake cron jobs. |
| Phase 4 — Chores & points | Fork Skylite-UX on GitHub. Add points column, completion tracking, chore recurrence. Biggest custom dev task. |
| Phase 5 — Gamification | Add streak tracking, badges DB table, celebration screen. Mostly front-end. |
| Phase 6 — Meal planner | Build the meal planning grid and recipe library. Wire up auto grocery-list generation. |
| Phase 7 — Message board | Add sticky-note posts with expiry. Low effort, high family value. |
| Phase 8 — Settings panel | Consolidate all configurable options into a parent-controlled settings page. |
| Phase 9 — Polish | Photo screensaver, motion wake sensor, QR code for phone access, UI polish. |
